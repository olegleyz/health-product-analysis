# Health Concierge

Personal health concierge agent for Telegram. Proactively checks in about workouts, nutrition, sleep, and recovery. Integrates Garmin, Oura, Strava, and Renpho data.

## Setup

```bash
cp .env.example .env
# Fill in your API keys and tokens in .env
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest
```
